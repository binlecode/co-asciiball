Killer use-case mockups

One frame set per use case. "Real" frames are the actual engine (ascii_sphere.Globe.render) at shipped/tunable defaults. "Conceptual" frames are a compositing prototype (not shipped) illustrating a proposed feature -- see each card's note.

01Live GeoIP + Network Traffic Visualization

Conceptual overlay -- packet destinations recolored with the heat-compositing model from Use Case 8; the engine has no GeoIP or event-flash feature today.

02AI Agent "Nervous System" -- Terminal Display Panel

Real engine output. --fill alone drives ghost -> solid; --speed (urgency) doesn't show in a still frame.

03Terminal Screensaver / Ambient Desk Ornament

Real engine output, shipped defaults.

04CI/CD "Build Planet" -- Status Pipeline as Porosity

Real engine output, matching the table in Use Case 4. Speed (spin rate) is a temporal cue, not visible in a still frame.

05Document Embed / PR Changelog Art

Real engine output. --glyph-source CHANGELOG.md -- the surface text really is the release notes, laid out in reading order across the globe.

06Terminal Activity-Wall Background

Real engine output, two rotation angles -- a tmux pane left running as ambient art.

07Programmatic Art Generator -- Export for Embedding

Real engine output at small radius, the shape a fastfetch logo or Neovim dashboard header would embed.

08Git Contributor Geo-Activity Heatmap

First two frames: the design problem from KILLER-USE-CASES.md -- painting heat as a flat color patch flattens the sphere (naive); compositing heat as hue/saturation ONLY, with lightness still riding the existing radial shade band, keeps the dome legible (HSL-composited). Third frame is the real pipeline: `git log --all` on THIS repo, 1 hot 15-degree band(s) out of 24 -- a one-author repo legitimately lights up one meridian, not a scattered map. Caveat: a UTC offset is a solar meridian, not a political timezone -- US Eastern (-04:00 in DST) resolves to ~-60 degrees longitude (mid-Atlantic), not the US East Coast; this is the accuracy ceiling of the zero-dependency default, not a bug.

09"Planet of Your Codebase" -- Git Activity Heatmap

Moon relief (real feature) as the closest shipped analogue for 'stale code erodes into gaps'; heat overlay (same compositing as Use Case 8) standing in for 'hot modules'.

10Honeypot / Security Monitoring Focal Point

Attack-source heat overlay (conceptual) plus a real high-void frame showing the far shell bleed through near-side gaps -- the 'through-Earth attack line' cue is structural (porosity), independent of color.
